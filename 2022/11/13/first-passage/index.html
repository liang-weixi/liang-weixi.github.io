如何在vscode中配置C语言的运行环境（2022版）#

索索引：当程序员的应该都知道vscode有多香，但是这个地表最强的编辑器对新手来说并不是那么友好，所以，此时上帝伸出了他的手说：“我来推你一把。”然后就有了这篇推文（迫真）

设备： 联想 Yoga 16s

系统： win11（22H2）

索引

1、前期文件下载

2、配置vscode

3、“Hello World”试验

1、 前期文件下载

  • 前往vs官网下载官网最新版的vscode

  • 进了官网点个下载就可以了，不是很麻烦

  • 然后再下载mingw64

记得下载的是mingw-18.0-without-git.exe！！！！！

  • 接下来只要将这俩玩意安装好就OK啦

2、配置vscode

  • 进入安装了mingw64的文件夹，单击文件夹上方的文件夹地址，Ctrl+C复制

  • 进入文件资源管理器，单击选中“此电脑”在上方的更多选项中单击属性，在新出现的页面中单击高级系统设置，在单击环境变量，在用户变量中选择Path，再选择编辑，新建，将刚才复制的地址粘贴进入框中。

  • 单击确定，再单击“系统属性”页面中的确定，此刻mingw64的环境变量就配置好啦。

  • 打开终端（cmd），输入

gcc -v
g++ -v

  • 终端会显示当前安装的gcc和g++的版本，显示成功即可

  • 打开新安装好的vscode，安装插件code runner 和C/C++，（可以多安装一个汉化插件Chinese），安装好后重启vscode

  • 重启后打开插件页，右键code runner后在拓展菜单中打开拓展配置，找到run in terminal和save file before run后勾选，设置完成。

  • 以同样的方式打开C/C++的拓展配置，找到compiler path，输入

~\bin\g++.exe

  • 找到cpp standard和c standrad，分别选择c++14和c11。

  • 找到include path，点击添加，输入

~\lib

  • 找到Intelli Sense Mode，选择gcc-x64

  • 设置完成，关闭设置页

3、“Hello World”试验

  • 单击左侧的资源管理器，选择一个位置新建一个文件夹用来装代码文件，再在此文件夹中新建一个Hello World.c文件，输入
#include<stdio.h>
int main()
{
   printf("Hello World!");
   return 0;
}

  • （不会还有人不会吧）

  • 右键后选择“Run Code”，在下端“终端”页面中见到输出“Hello World！”即为成功。

extra：配置调试器

  • 写个长一点的代码，打上断点

  • 在上方菜单中选择运行，启动调试，在弹出的选择菜单中选择C++（GDB/LLDB）,g++.exe，见到左侧菜单出现“运行和调试”页面即为成功。
